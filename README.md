# pipeline-kubernetes
Pipeline orchestrated by kubernetes

# Usage
git clone  
```git clone git@github.com:open-devops/pipeline-kubernetes```

pipeline init  
```cd pipeline-kubernetes; sh pipeline init```

pipeline start up  
```sh pipeline up```

:pipeline status confirmation  
```sh pipeline status```

:pipeline stop  
```sh pipeline stop``````
